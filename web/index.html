Why MargaritaWP

Everything you need for margarita maths.

A focused set of tools for calculating, customising and embedding margarita recipes on WordPress.

⚖️
Ratio-first recipe scaling
Select a built-in preset or define your own ratio, then scale by drink count, pitcher size or party planning needs.
📐
Flexible units
Work in ml, oz, shots or nips. MargaritaWP keeps the quantities consistent across serving sizes.
🍓
Flavour variations
Add spicy, mango, watermelon, strawberry, coconut or virgin variations without rebuilding the whole recipe.
📦
Compact widget mode
Use the full calculator on landing pages, or embed a smaller widget inside individual recipe posts.
🔒
No third-party services
Calculations run inside your WordPress site with local AJAX/REST handlers. No hosted calculator service required.
🧪
ABV and standard drink estimates
Estimate ABV, alcohol grams and standard drinks using AU, UK or US definitions for recipe planning.
⚙️
WordPress settings screen
Choose default units, presets, ABV display and custom recipe ratios from the WordPress settings screen.
🔓
Open source
Inspect the code, fork the plugin or adapt it for your own WordPress recipe workflow.

How it works

From install to embedded calculator.

Add a full calculator with a shortcode, use the Gutenberg block, or place a compact widget inside a recipe post.

1
Install the plugin
Download MargaritaWP from GitHub and activate it on your WordPress site.
2
Configure defaults
Set default units, presets, ABV display and custom ratios in the WordPress settings screen.
3
Embed anywhere
Drop a shortcode into any post or page. Give readers a calculator they can actually use.
Simple to embed

Built for WordPress.

Drop these shortcodes into any WordPress post or page. Full documentation is available in the GitHub repository.

Basic shortcode Page / Post
[margarita_measurements]
Preset example Page / Post
[margarita_measurements
  preset="tommys"
  drinks="4"
  unit="ml"]
Compact widget Recipe Post
[margarita_widget
  preset="classic"
  drinks="2"
  align="right"]
Use cases

Who it's for

Turn static cocktail recipes into interactive recipe tools.

🍹 Cocktail bloggers
Give readers a calculator they can actually use. Let them scale a recipe from one drink to a batch without leaving the page.
🍸 Bars and venues
Add practical recipe calculators to event pages, menu pages or training content.
🎓 Hospitality schools
Help students understand ratios, units, batch scaling and alcohol estimates in a WordPress-based learning site.
📰 Recipe and lifestyle sites
Use a shortcode, Gutenberg block, AJAX form and REST endpoints without relying on a third-party hosted calculator.
