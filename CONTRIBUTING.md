# Contributing to Margarita Measurements

Thanks for your interest in improving this plugin!

## How to Contribute
1. Fork and clone your fork.
2. Create a feature branch:
   ```bash
   git checkout -b feature/my-feature
   ```
3. Make clear, small commits.
4. Add tests for logic changes (if applicable).
5. Run coding standards:
   ```bash
   phpcs --standard=WordPress .
   ```
6. Push and open a Pull Request.

## Code Style
- WordPress Coding Standards.
- Escape output; sanitize input.
- Use nonces for form submissions.

## Reporting Issues
Open a GitHub issue with WP + PHP versions, steps to reproduce, and expected vs actual behavior.
